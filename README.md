Learnix is an AI-driven study tool designed to transform raw academic content into structured, exam-focused learning material. It enables students to convert text into quizzes, notes, and revision resources, all organized in a centralized library for efficient studying.

---

## Features
- AI-generated quizzes from academic content  
- Intelligent structuring of study material  
- Centralized library for revision resources  
- Clean, distraction-free learning flow  

---

## Tech Stack
- **Frontend:** React  
- **Backend:** Node.js  
- **AI:** Large Language Models (LLMs)  
- **APIs:** AI content generation APIs  

---

## System Focus
- Accurate transformation of academic content  
- Exam-oriented output instead of generic summaries  
- Simple UX designed for repeated daily use  

---

## Use Cases
Learnix is ideal for students who want to:
- Save time creating study material  
- Revise efficiently instead of rereading textbooks  
- Practice using AI-generated, topic-focused quizzes
